# 용돈통장

아이가 해낸 만큼 쌓이는 우리 집 용돈 통장. 설치 없이 브라우저에서 돌아가는 PWA입니다.

- 기록은 **각자 기기 안에만** 저장됩니다. 서버로 나가지 않습니다.
- 인터넷 없이도 열립니다.

## GitHub Pages로 올리기

1. 이 폴더의 파일을 저장소 **최상단**에 올립니다. (`index.html`이 루트에 있어야 합니다)
2. 저장소 → **Settings → Pages**
3. Source를 **Deploy from a branch**, 브랜치는 `main`, 폴더는 `/ (root)` 로 지정하고 저장
4. 1~2분 뒤 `https://<아이디>.github.io/<저장소이름>/` 에서 열립니다

## 고친 뒤에 할 일

`sw.js` 맨 위 버전을 올리세요. 안 그러면 기기에 남은 옛 캐시가 계속 나옵니다.

```js
const VERSION = "v1";  // -> "v2"
```

## 폴더 구성

```
index.html            앱 전체 (화면 + 로직)
sw.js                 오프라인 캐싱
manifest.webmanifest  앱 이름, 아이콘, 실행 방식
icons/                아이콘 4종
.nojekyll             Jekyll 빌드 끄기
```

## 백업

설정 화면 맨 아래에서 JSON 파일로 내려받을 수 있습니다. 브라우저 기록을 지우면
저장된 기록도 함께 사라지니 한 달에 한 번은 받아 두세요.
